# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and harassment-free experience for everyone participating in the AdrPlus community.

## Expected Behavior

- Be respectful and constructive in discussions and reviews.
- Assume positive intent and provide actionable feedback.
- Focus on technical topics and project improvement.
- Help create an inclusive environment for contributors of all backgrounds and experience levels.

## Unacceptable Behavior

- Harassment, discrimination, hate speech, or personal attacks.
- Trolling, insulting comments, or disruptive behavior.
- Publishing private information without explicit permission.
- Any conduct that is inappropriate in a professional open source setting.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and other community interactions.

## Reporting

If you experience or witness unacceptable behavior:

1. Open a private maintainer contact through GitHub (repository maintainers).
2. If needed, use GitHub moderation/reporting tools for immediate concerns.

All reports will be reviewed promptly and handled with discretion.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.
They may remove, edit, or reject comments, commits, code, issues, and other contributions that violate these guidelines.
